# Employee Management and Payroll System

## Overview
Employee Management and Payroll System is a desktop application developed using Java Swing, JDBC, and MySQL. It provides role-based access for Admin, HR, and Employees to manage employee information efficiently.

## Tech Stack
- Java 17
- Java Swing
- JDBC
- MySQL
- Eclipse IDE
- Git
- GitHub

## Features
- Secure User Authentication
- Role-Based Login (Admin, HR, Employee)
- Employee Registration
- MySQL Database Integration
- JDBC Connectivity
- Object-Oriented Programming (OOP) Design
